# Healthcare Appointment Analysis

## Project Overview
This project performs an in-depth data analysis of healthcare appointments to uncover valuable business insights that can improve administrative and operational efficiency. We evaluate patient no-show rates, top reasons for visits, and provider workloads using Python and specialized data analysis libraries.

## Setup and Installation
To run the analysis locally, ensure you have Python 3 and Jupyter Notebook installed.

1. Clone or download this repository.
2. Install the necessary dependencies (pandas, matplotlib, seaborn):
   ```bash
   pip install pandas matplotlib seaborn jupyter notebook openpyxl
   ```
3. Open the repository directory in your terminal or command prompt.
4. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

## Files in this Project
- **`Healthcare_Appointments_Analysis.ipynb`**: The main Jupyter Notebook where the full analysis, data cleaning, and data visualization happens. You can execute each cell to see the charts and output.
- **`Healthcare Appointments.xlsx`**: The raw data file used for the analysis containing over 10,000 appointment records.
- **`generate_health_nb.py`**: A python script used to procedurally generate the Notebook. (Optional to run)

## Execution
Open the `.ipynb` file in your Jupyter environment. Run all cells from top to bottom.
The analysis includes:
1. Data Cleaning
2. Handling null values and optimizing column data-types.
3. Visualization charts such as provider appointment distribution.
4. Extracted business insights and operational recommendations.
